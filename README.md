# Cheque-Management-System
A modern cheque note management system built with HTML, CSS, and JavaScript featuring search, filtering, sorting, and local storage support.

Smart Cheque Note Management System

Smart Cheque Note System is a modern web-based cheque record management application designed for small businesses and financial tracking.

🚀 Features

Add and manage cheque records

Search by cheque number, name, bank, description

Filter by Today, Week, Month, Year

Sort by multiple fields

Delete confirmation modal

Highlight past dates

Responsive modern UI

LocalStorage-based persistent data

Print-ready table format

🛠 Built With

HTML5

CSS3 (Responsive Design + Modern UI)

JavaScript (ES6)

Font Awesome Icons

Browser LocalStorage API

No backend or database required. All records are securely stored inside the browser.
